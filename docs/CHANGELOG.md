## 2025-01-27 - Рождественская тема и страница загадывания желаний

### Новые функции

1. **Рождественская тема**
   - Добавлен компонент `Snowflakes.tsx` для анимации снежинок
   - Анимация снежинок с настраиваемыми параметрами (количество, скорость, размер, цвет)
   - Эффекты ветра и вращения снежинок
   - Управление темой через админ-панель (`/admin/settings`)
   - Настройка хранится в таблице `site_settings` в базе данных
   - Хук `useChristmasTheme()` для проверки состояния темы

2. **Страница загадывания желаний на 2026 год**
   - Новая страница `/wish-2026` для создания свечей с желаниями на новый год
   - Форма для записи заголовка и подробного описания желания
   - Выбор длительности свечи (1 день, 1 неделя, 1 месяц)
   - Опция анонимности заголовка
   - Специальный тип свечи `wish_2026`

3. **Блок "Прощаемся с 2025 годом"**
   - Компактный блок на главной странице под hero-секцией
   - Отображается только при включенной рождественской теме
   - Кнопка для перехода на страницу загадывания желания

4. **Админ-панель: Настройки сайта**
   - Новый раздел `/admin/settings` для управления настройками сайта
   - Переключатель рождественской темы
   - API endpoint `/api/admin/settings` для получения и обновления настроек

### Технические изменения

- Создана таблица `site_settings` в Supabase для хранения настроек сайта
- Добавлен SQL миграционный скрипт `supabase-christmas-theme-setup.sql`
- Реализована проверка прав администратора в API endpoint настроек
- Удален компонент `ChristmasDecorations.tsx` (заменен на более минималистичный дизайн)
- Обновлена структура профиля пользователя (кнопка "Настройки" перенесена в блок "Админ-панель")

### Изменения в документации

- Обновлен `README.md` с информацией о новых функциях
- Обновлен `docs/features/CHRISTMAS_THEME.md` с актуальной информацией
- Добавлена документация по API endpoint `/api/admin/settings`
