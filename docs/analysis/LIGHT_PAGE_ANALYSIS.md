# Анализ страницы /light и предложения по улучшению

## Текущее состояние:

### Структура:
1. **Заголовок** - простой заголовок и описание
2. **CandleHero** - динамический HERO с свечой и текстом по типу
3. **Форма** - выбор типа, поля title/message, длительность, анонимность
4. **Модалка успеха** - после создания свечи

## Выявленные проблемы:

### 1. Валидация ⚠️
- **Нет ограничений на длину** title и message
- Нет счетчиков символов
- Нет предупреждений при превышении лимита
- Нет валидации на клиенте перед отправкой

### 2. UX улучшения
- Нет визуальной обратной связи при вводе
- Нет подсказок/примеров для полей
- Можно улучшить визуальное оформление формы
- Нет индикации прогресса заполнения

### 3. Визуальные улучшения
- Форма может быть более выразительной
- Можно добавить больше визуальных акцентов
- Улучшить spacing и типографику

### 4. Функциональность
- Нет предпросмотра свечи перед созданием
- Нет сохранения черновика
- Нет быстрых шаблонов

## Предлагаемые улучшения:

### Приоритет 1 (Критично):
1. **Валидация форм** - ограничения длины, счетчики символов
2. **Улучшение UX** - визуальная обратная связь, подсказки

### Приоритет 2 (Важно):
3. **Визуальные улучшения** - более выразительная форма
4. **Предпросмотр** - показ как будет выглядеть свеча

### Приоритет 3 (Желательно):
5. **Шаблоны** - быстрые шаблоны для создания
6. **Сохранение черновика** - localStorage для незавершенных форм
