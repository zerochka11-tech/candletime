# Полный анализ главной страницы и единая стилистика

## Текущая структура страницы:

1. **HERO** - темный градиент, заголовок, CTA, анимированная свеча
2. **STATS** - 3 карточки со статистикой
3. **WHAT IS THIS** - описание с тремя пунктами
4. **HOW IT WORKS** - 3 шага
5. **CTA** - финальный призыв

## Единая стилистика проекта:

### Визуальные элементы:
- **Градиенты:** `from-slate-900 via-slate-900 to-slate-950`
- **Декоративные градиенты:** `from-amber-500/5 via-transparent to-indigo-500/5`
- **Цветовые акценты:** amber, emerald, indigo, sky, rose
- **Скругления:** `rounded-2xl`, `rounded-3xl`
- **Тени:** `shadow-sm`, `shadow-md`, `shadow-lg`
- **Границы:** `border border-slate-200/70`

### Интерактивность:
- **Hover эффекты:** `hover:-translate-y-0.5`, `hover:-translate-y-1`
- **Масштабирование:** `group-hover:scale-110`
- **Градиенты при hover:** `bg-gradient-to-br from-{color}-50/50`
- **Тени при hover:** `hover:shadow-md`, `hover:shadow-lg`

### Типографика:
- **Заголовки:** `text-xl font-bold md:text-2xl`
- **Описания:** `text-sm leading-relaxed md:text-base`
- **Мелкий текст:** `text-xs`
