# Анализ страницы /candles и предложения по улучшению

## Текущее состояние:

### Структура:
1. **Заголовок** - название и количество свечей
2. **Список свечей** - карточки с информацией о свечах
3. **Loading state** - простой текст
4. **Empty state** - сообщение когда нет свечей

### Функциональность:
- Загрузка активных свечей (expires_at > now)
- Сортировка по дате создания (новые сначала)
- Лимит 50 свечей
- Кликабельные карточки (ведут на /candle/[id])
- Hover-эффекты

## Выявленные проблемы:

### 1. Отсутствие фильтрации ⚠️
- Нет фильтра по типу свечи
- Нет поиска по названию/сообщению
- Нет фильтра по дате создания

### 2. Пагинация и загрузка ⚠️
- Только лимит 50 свечей
- Нет пагинации
- Нет "загрузить еще"
- Нет бесконечной прокрутки

### 3. Сортировка ⚠️
- Только сортировка по дате создания
- Нет сортировки по:
  - Времени до погасания
  - Типу свечи
  - Популярности

### 4. UX улучшения
- Простой loading state (можно улучшить)
- Нет обновления в реальном времени
- Нет индикации свечей, которые скоро погаснут
- Можно улучшить визуальное оформление

### 5. Визуальные улучшения
- Можно добавить больше визуальных акцентов
- Улучшить типографику
- Добавить иконки и визуальные маркеры

## Предлагаемые улучшения:

### Приоритет 1 (Критично):
1. **Фильтр по типу свечи** - быстрые кнопки для фильтрации
2. **Поиск** - поиск по названию и сообщению
3. **Улучшенный loading state** - skeleton loading вместо текста

### Приоритет 2 (Важно):
4. **Пагинация** - "загрузить еще" или бесконечная прокрутка
5. **Сортировка** - выбор способа сортировки
6. **Индикация скоро погаснущих** - визуальное выделение

### Приоритет 3 (Желательно):
7. **Обновление в реальном времени** - автоматическое обновление списка
8. **Статистика** - количество по типам
9. **Визуальные улучшения** - более выразительный дизайн
