# Анализ страницы /candle/[id] и предложения по улучшению

## Текущее состояние:

### Структура:
1. **Навигация** - кнопка "Назад к свечам"
2. **Карточка свечи** - тип, статус, дата, заголовок, сообщение, время
3. **Кнопки действий** - поделиться, копировать ссылку, зажечь свою свечу
4. **Loading state** - простой текст
5. **Error state** - сообщение об ошибке

### Функциональность:
- Загрузка свечи по ID
- Отображение полной информации о свече
- Поделиться (Web Share API)
- Копирование ссылки
- Определение статуса (активна/погасла/погашена)

## Выявленные проблемы и возможности улучшения:

### 1. Loading state ⚠️
**Проблемы:**
- Простой текст "Загружаем свечу…"
- Нет визуальной обратной связи
- Можно улучшить UX

**Улучшения:**
- Skeleton loading с формой карточки
- Анимированные плейсхолдеры
- Более информативный loading

### 2. Визуализация свечи ⚠️
**Проблемы:**
- Нет визуального представления свечи
- Можно добавить анимированную свечу
- Нет индикации "горит" для активных свечей

**Улучшения:**
- Анимированная свеча (если активна)
- Визуальная индикация статуса
- Цвета свечи по типу

### 3. Обновление времени ⚠️
**Проблемы:**
- Время "Осталось" не обновляется в реальном времени
- Статичное отображение

**Улучшения:**
- Автоматическое обновление каждую минуту
- Live-счетчик оставшегося времени
- Визуальная индикация приближения к погасанию

### 4. SEO и шаринг ⚠️
**Проблемы:**
- Нет мета-тегов для Open Graph
- Плохие превью при шаринге в соцсетях
- Нет динамических мета-тегов

**Улучшения:**
- Open Graph мета-теги
- Twitter Card мета-теги
- Динамические title и description

### 5. UX улучшения
**Проблемы:**
- Нет хлебных крошек
- Нет навигации к следующей/предыдущей свече
- Можно улучшить визуальное оформление

**Улучшения:**
- Хлебные крошки (Главная > Свечи > Название свечи)
- Кнопки навигации (если есть похожие)
- Улучшенные hover-эффекты

### 6. Визуальные улучшения
**Проблемы:**
- Можно добавить больше визуальных акцентов
- Улучшить типографику
- Добавить декоративные элементы

**Улучшения:**
- Градиенты и тени
- Улучшенная типографика
- Декоративные элементы

### 7. Дополнительные функции
**Проблемы:**
- Нет похожих свечей
- Нет истории просмотров
- Нет возможности редактировать (для владельца)

**Улучшения:**
- Похожие свечи (по типу)
- История просмотров (localStorage)
- Редактирование для владельца (если авторизован)

## Предлагаемые улучшения:

### Приоритет 1 (Критично):
1. **Skeleton loading** - улучшенный loading state
2. **Анимированная свеча** - визуализация для активных свечей
3. **Live-обновление времени** - автоматическое обновление "Осталось"

### Приоритет 2 (Важно):
4. **SEO мета-теги** - Open Graph, Twitter Cards
5. **Хлебные крошки** - улучшенная навигация
6. **Визуальные улучшения** - градиенты, тени, типографика

### Приоритет 3 (Желательно):
7. **Похожие свечи** - рекомендации
8. **История просмотров** - localStorage
9. **Редактирование** - для владельца свечи
