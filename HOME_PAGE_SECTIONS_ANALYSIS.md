# Анализ блоков "Что это?" и "Как это работает" на главной странице

## Текущее состояние:

### Блок "Что это?" (строки 252-295):
- Простая белая карточка с border
- Заголовок и описание
- Список из 3 пунктов с иконками в цветных квадратах
- Простые hover эффекты (hover:bg-slate-50)

### Блок "Как это работает" (строки 297-361):
- Белая карточка с ring
- Заголовок и описание
- 3 колонки с шагами (1, 2, 3)
- Номера в черных квадратах
- Эмодзи рядом с номерами
- Простые hover эффекты

## Стилистика проекта:

### Используемые элементы:
- Градиенты: `from-slate-900 via-slate-900 to-slate-950`
- Цветовые акценты: amber, emerald, indigo, sky, rose
- Анимации: hover:-translate-y-1, animate-pulse
- Декоративные элементы: градиенты, blur эффекты
- Карточки: rounded-2xl/3xl, shadows, borders
- Иконки: эмодзи в цветных контейнерах

## Предложения по улучшению:

### 1. Блок "Что это?"
**Проблемы:**
- Слишком простой дизайн
- Нет визуальной связи с общей стилистикой
- Иконки выглядят базово

**Улучшения:**
- Добавить градиентный фон или цветовые акценты
- Улучшить визуальную иерархию
- Добавить декоративные элементы
- Улучшить hover эффекты
- Сделать иконки более выразительными

### 2. Блок "Как это работает"
**Проблемы:**
- Черные квадраты с номерами выглядят тяжело
- Нет визуальной связи между шагами
- Простые hover эффекты

**Улучшения:**
- Заменить черные квадраты на градиентные/цветные
- Добавить визуальную связь между шагами (линия, стрелки)
- Улучшить визуальную иерархию
- Добавить больше интерактивности
- Улучшить типографику
